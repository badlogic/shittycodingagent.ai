Slash Commands

Command Description
/model Switch models mid-session
/thinking Adjust thinking level (off/minimal/low/medium/high)
/branch Create new conversation branch
/resume Switch to a different session
/compact Manually compact conversation context
/export Export session to HTML
Philosophy

Opinionated by design

No MCP

CLI tools over protocol overhead. Any executable with a README works.

No Sub-Agents

Spawn pi via bash if needed. No context loss from delegation.

No Permission System

Full filesystem access. Run in a container if uncomfortable.

No Planning Mode

Tell the agent to think. Write plans to files for persistence.

Usage

CLI Examples

examples
# Interactive mode
$ pi
# Non-interactive with prompt
$ pi -p "List all .ts files in src/"
# Continue last session
$ pi -c
# With file context
$ pi @code.ts "Review this code"
# Specific model
$ pi --provider openai --model gpt-4o
