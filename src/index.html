What you get

The boring stuff that actually matters

4 tools
read, write, edit, bash. That's it. No MCP, no tool zoo.
Multi-provider
Anthropic, OpenAI, Google, Mistral, Groq, xAI, OpenRouter, Ollama. Add your own.
/model
Switch models mid-conversation. Because one size never fits all.
Sessions
pi -c continues. pi -r picks from history. /branch for bad ideas.
Compaction
Long conversations get summarized so you don't hit context limits mid-thought.
AGENTS.md
Project instructions loaded automatically. Global or per-directory.
Custom commands
Markdown files become slash commands. Reusable prompts without copy-paste.
Skills
On-demand instruction files. Agent loads them when relevant.
Hooks
TypeScript modules for permission gates, checkpoints, or injecting messages.
Themes
Dark, light, or roll your own. Hot reload included.
Bash mode
Prefix with ! to run commands. Output goes into context.
RPC mode
JSON over stdin/stdout. Embed it in your own tools.
Philosophy

What we didn't build

No MCP

CLI tools work fine. Always have.

No sub-agents

Context doesn't survive delegation.

No permission popups

They're security theater anyway.

No "plan mode"

Just ask it to think first.
